📞 CallCore — Call Center Management System
A full-featured, browser-based Call Center Database Management System built with pure HTML, JavaScript, and SQLite (via WebAssembly). No backend, no installation, no dependencies to install — just open index.html and it works.

✨ Features

📊 Dashboard

Live KPI cards — Calls Today, Active Agents, Open Tickets, Avg Handle Time

14-day call volume trend line chart

Call type distribution donut chart

Recent calls table & urgent ticket panel

Top agents leaderboard

📞 Call Logs

Log new calls with customer, agent, type, duration, outcome & notes

Filter by call type, outcome, and department

Full-text search across customers and agents

Delete call records

👤 Agents

Card view and table view toggle

Add, edit, and deactivate agents

Agent profile modal with full call history and KPI stats

Resolution rate progress bar per agent

🧑‍💼 Customers

Full customer directory with tier classification (Standard / Premium / VIP / Enterprise)

Add and remove customers

Customer history modal — complete call log + linked tickets

🎫 Support Tickets

Kanban board view (Open → In Progress → Resolved)

List view toggle

Create tickets with priority (Low / Medium / High / Critical) and category

One-click status updates and ticket deletion

📈 Analytics

Calls by department (bar chart)

Call outcome resolution rates (pie chart)

Avg handle time by department (horizontal bar chart)

Top 10 agent leaderboard with resolution rates

Outcome summary with animated progress bars

🗄️ SQL Console

Live SQL query editor (press Ctrl + Enter to run)

9 built-in quick query presets

Results displayed as formatted tables

Full database schema browser
